# WebApp-Java-RestFul-web-services

  ### Used Java, JDBC, MySQL, jQuery, HTML, CSS
  ### Developed RestFul webservices: 

  * getMovies
  * getMoviesByName
  * getMoviesByTheater
  * getMoviesByRatings
  * getMoviesByGenre
  * insertNewMovie
  * updateNewMovieByName
  * deleteMovieByName

    String  director;
    String  production;
    String  movRating;
    String  overview;
    String  theaterName;
    String  address;
